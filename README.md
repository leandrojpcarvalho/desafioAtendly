# desafioAtendly

Ola, para executar todos procedimentos, tenha certeza que esta na raiz do monorepo.

## Instalaçao de pacotes:

```bash
npm run install
```

## Para subir a aplicaçao:

```bash
npm run dev
```

## Para executar testes:

```bash
npm run test
```

## Para verificar a cobertura de testes:

```bash
npm run coverage
```
